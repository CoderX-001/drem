Getting started with drem

At drem, we care about your safety so we created our API structure with top security in mind.

To get started with drem, you have to follow these steps:

1

Create a drem account here. Creating an account is very easy. Just follow the instructions and you are good to go.

2

Generate your api key. Api keys contains informations about a specific user, such as the developer who owns the access token, the type of app the developer owns and other information which helps us to make the API easy-to-use for you.

To create an access token, you have to specify the type of app you want to create and the environment you want to work with (development or production).

3

Once you have generated your api key, you can now access the api from your frontend without having to create a backend server for your backend work.
